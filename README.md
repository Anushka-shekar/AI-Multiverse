ASSIGNMENT 1 - This Streamlit application collects a user's name and message, validates the input, displays a success or error message, and estimates the approximate token usage based on the message length. It demonstrates basic user input handling, conditional logic, and interactive UI development using Streamlit. ( app1.py )
 
ASSIGNMENT  2 - Basically, in the AI multiverse setup, you just setup the Google Gemini SDK once. Then in your prompt, you give the AI different personas like "act as a comedian" or "act as a serious coach". After that, you chat with whichever persona you want and the same AI replies in totally different tones based on that role.
( app2.py )

ASSIGNMENT 3 -  The Multiverse of Chatbots is an interactive Streamlit application powered by Google Gemini 2.5 Flash. Users can choose from multiple fictional characters and personalities, then have engaging conversations while each chatbot responds in character. The app features a cosmic glassmorphic UI, customizable response length, persistent chat history, and secure API key management using environment variables or Streamlit Secrets.( app3.py )

ASSIGNMENT 4 - 🎨 **AI Image Generator** is a Streamlit-based application that generates images from text prompts using the Pollinations AI image generation service. Users can select art styles, customize image dimensions, enable Magic Enhance for improved prompts, generate surprise images, and download the generated artwork. ( app4.py )
