# Assignment 4 - AI Image Generator

🎨 AI Image Generator is a Streamlit-based application that generates images from text prompts using the Pollinations AI image generation service.

Users can select different art styles, customize image dimensions, enable Magic Enhance, generate surprise images, and download the generated artwork.
